# DATA_STRUCTURE_ALGORITHM_05
## Handle: _MUMINUL__ISLAM___



## Description

**Data_Structure_Algorithm_05** is a Near-Production-Ready Example Project as a Portfolio.

## Features

-It Contain Stack,Queue,LRU Cache Implemented with Python Language.
-It have Operations Such As-Push, POP, PEEK, isEmpty, Size, Enqueue, Dequeue
-The Cache I Implemented Here is LRU CACHE(Least Recently Used)
-The Cache Implemented with a Doubly Linked List and HashMap
-LRU Cache GET, PUT Operation Will be Implemented Later.

## Tech Stack

**Programming Language**: python 3.12+
**Testing**: pytest
**Module**:(GC) Garbage Collection
**Package Management**: pip

pip install -r requirements.txt
